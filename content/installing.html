Installing from a tar-file release

You can download tar-file releases from the links in the menu or by visiting the download area.

Unpack and change directory with:

tar xfz firehol-version.tar.gz
cd firehol-version

Options for the configure program can be seen in the INSTALL file and by running:

./configure --help

To build and install taking the default options:

./configure && make && sudo make install

Alternatively, just copy the sbin/firehol.in and sbin/fireqos.in files to where you want them.

Note that packages are available for most distributions but they will not always offer the latest version.
