Quick Start

First install the software, either from your distribution, from a tar-file or from the git repository.

FireHOL

To gain a greater understanding and for step-by-step help building a custom firewall, see the tutorials. To have FireHOL guess the configuration for your machine, simply run: 

firehol helpme > /tmp/firehol.conf

The purpose of the <%= fh_manref('firehol','helpme') %> feature is to give you a configuration file that you can modify to get an operational firewall quickly, especially if your firewalling and iptables knowledge is limited. This feature does not stop or alter the running firewall of your machine.

The output file must be edited before you use it, following the embedded instructions.

FireQOS

FireQOS does not have any mechanism to automatically detect and classify your traffic.

This FireQOS tutorial is a step by step guide for setting up traffic control on a system. The steps followed will allow you to apply QoS without disrupting your current service or traffic or set up traffic monitoring without traffic shaping.
