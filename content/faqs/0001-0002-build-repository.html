--- group: 0001-Installation title: 0002-How do I build from the repository? kind: faq ---

You will need xsltproc, dblatex, DocBook XML, DocBook XSL Stylesheets, GNU autoconf and GNU automake.

When building from the Git repository you can create the Makefiles and enable building the manual using these commands: 

./autogen.sh
./configure --enable-maintainer-mode
