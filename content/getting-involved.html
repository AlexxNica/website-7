Getting Involved

If you are an experienced user willing to help others, please join up to the support mailing list and offer advice where you can.

Please consider introducing yourself on the development mailing list (archive). Alternatively drop an existing project member an email.

Or, just fork the repository and get coding. See this FAQ about building from the repository.

Help on the project will be welcomed in any area - development, testing, infrastructure, documentation, web design or anywhere else you have the time and inclination to offer it.
