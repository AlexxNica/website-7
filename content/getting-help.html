Getting Help

If you need help setting up or running FireHOL, the best thing to do is join the support mailing list where you can get help from the authors and other experienced users of FireHOL.

The list has a searchable support archive which you can use to check to see if your question has been answered before.

We will also try to help you if you create an entry tagged help in the FireHOL issue tracker on GitHub.
