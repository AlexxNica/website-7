Distribution Packages

Packages are available for most distributions. If you cannot find one for your distribution or you want a more recent release, see the standard installation instructions.

Debian/Ubuntu

To install and enable from system packages: 

apt-get update
apt-get install firehol
vi /etc/default/firehol

Search for: Debian packages, Ubuntu packages and see also Ubuntu Launchpad.

Arch Linux

There is an AUR for the development version.

OpenWRT

There is no package available, but we have a page with comprehensive instructions for installing FireHOL and FireQOS on OpenWRT.

Others

If you know of any other FireHOL packages and can provide simple install instructions, please let me know and I will add them to this page.
