Distribution Packages

Packages are available for most distributions. If you cannot find one for your distribution or you want a more recent version, you can download a tar-file from the downloads area. Once unpacked, you can install it with:

./configure && make && sudo make install

Options can be seen in the INSTALL file and by running:

./configure --help

Alternatively, just copy the sbin/firehol.in file to where you want it.
